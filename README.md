A console-based Python implementation of **Kaun Banega Crorepati (KBC)**.

## How the Game Works
- User enters 5 questions with 4 options each
- Correct answer is predefined
- Player selects options (A/B/C/D)
- Prize money increases for every correct answer
- Game ends on the first wrong answer
